# Podium Cars

## Descripción

**Podium Cars** es un proyecto diseñado para facilitar la gestión y organización de un concesionario o vendedor de vehículos. El sistema permite administrar de forma rápida y eficiente diferentes procesos relacionados con la compra, venta e importación de automóviles, proporcionando una estructura clara para el control del inventario y las operaciones del negocio.

## Funcionalidades

- **Gestión de inventario de vehículos**
  - Registro de vehículos de segunda mano.
  - Visualización de la lista de vehículos disponibles.
  - Eliminación de vehículos cuando se realiza una venta.

- **Registro de modificaciones**
  - Posibilidad de registrar modificaciones internas como:
    - Stage de rendimiento
    - Cambios en la transmisión
    - Suspensión
    - Sistema de escape
  - Registro de modificaciones externas como:
    - PPF
    - Alerones
    - Piezas en fibra de carbono
    - Cambio de piezas estéticas o estructurales

- **Gestión de importaciones**
  - Registro de la marca y modelo del vehículo a importar.
  - Cálculo automático de una fecha estimada de llegada basado en la fecha actual.
  - Permite al vendedor informar al cliente sobre el tiempo aproximado de entrega.

## Objetivo del proyecto

El objetivo principal de **Podium Cars** es optimizar la organización interna de un concesionario, mejorar el seguimiento del inventario de vehículos y proporcionar una herramienta sencilla para gestionar las operaciones diarias del negocio.
